# Usage

```
$ git clone git@github.com:mochizukikotaro/docker-rails-mysql.git
$ cd docker-rails-mysql
$ docker-compose up
```

```
$ docker-compose exec app bash
# yarn install
# bundle install
# rails db:create
# rails s -b 0.0.0.0
```

You can access `localhost:3000`
